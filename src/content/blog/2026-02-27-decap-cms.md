**Decap CMS** (anteriormente conocido como Netlify CMS) es un sistema de gestión de contenido de código abierto basado en Git. A diferencia de CMS tradicionales como WordPress que usan bases de datos, Decap CMS guarda todo el contenido directamente en tu repositorio de Git (GitHub, GitLab, etc.) en forma de archivos de texto (Markdown, JSON, YAML).

![](/src/assets/images/uploads/download-removebg-preview_-23-.png "imagen")

**¿Cómo funciona?**

* Los editores acceden a una interfaz amigable en `/admin` para crear y editar contenido.
* Cada cambio genera un commit en el repositorio.
* Se integra con flujos de trabajo Git (pull requests, revisiones, etc.).

**Ventajas principales:**

* ✅ **Sin base de datos** – todo vive en tu repositorio.
* ✅ **Los editores no necesitan saber Git** – trabajan con una interfaz visual.
* ✅ **Ideal para sitios estáticos** (Astro, Hugo, Gatsby, etc.).
* ✅ **Open source y auto-hospedable.**

En nuestro caso, lo usaremos para que los administradores puedan crear y editar posts del blog sin tocar código, mientras todo se sincroniza automáticamente con nuestro repositorio en GitHub.\
